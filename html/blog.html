Question-1: What are the different ways to select an element in the DOM?

Ans:

DOM-এ একটি উপাদান নির্বাচন করার বিভিন্ন উপায় রয়েছে:
document.getElementById("id") – আইডি দ্বারা এলিমেন্ট সিলেক্ট করে। document.getElementsByClassName("class") – ক্লাস নাম দ্বারা এলিমেন্ট সিলেক্ট করে। document.getElementsByTagName("tag") – (এই অংশ বাদ দেওয়া হলো) document.querySelector("selector") – CSS সিলেক্টর ব্যবহার করে প্রথম মিল থাকা এলিমেন্ট সিলেক্ট করে। document.querySelectorAll("selector") – CSS সিলেক্টর ব্যবহার করে সব মিল থাকা এলিমেন্ট সিলেক্ট করে।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:

innerHTML, innerText এবং textContent এর মধ্যে পার্থক্য:
innerHTML – HTML ট্যাগসহ কন্টেন্ট রিটার্ন বা সেট করতে পারে। innerText – শুধুমাত্র দৃশ্যমান সেট করে, হিডেন টেক্সট নেয় না। textContent – সম্পূর্ণ টেক্সট রিটার্ন বা সেট করে, হিডেন টেক্সটও অন্তর্ভুক্ত হয় তবে HTML প্রসেস করে না।

Question-3: What is event delegation in the DOM?

Ans:

Event Delegation হলো একটি কৌশল যেখানে প্যারেন্ট এলিমেন্টে ইভেন্ট লিসেনার সেট করা হয়, যাতে তার চাইল্ড এলিমেন্টগুলোর ইভেন্ট হ্যান্ডল করা যায়।

Question-4: What is event bubbling in the DOM?

Ans:

Event bubbling মানে হলো, যখন কোনো child element-এ ক্লিক করা হয়, ইভেন্টটি প্রথমে সেখানে ঘটে, তারপর তার parent, grandparent হয়ে উপরের দিকে যায়। এর ফলে parent elements তাদের child-এ ক্লিকের প্রতিক্রিয়া জানাতে পারে।

Question-5: How do you create, add, and remove elements using JavaScript?

Ans:

JavaScript-এর মাধ্যমে ডকুমেন্টে এলিমেন্ট তৈরি করতে document.createElement() ব্যবহার করা হয়, নতুন এলিমেন্ট যোগ করতে appendChild() ব্যবহার করা হয় এবং এলিমেন্ট মুছতে remove() ব্যবহার করা হয়। এটি ওয়েবপেজকে ডায়নামিকভাবে পরিবর্তন করতে সাহায্য করে।
